# Rekonekt PHP SDK

Library to help working with Rekonekt API.

Usage with composer
----

1. Add this package to your composer.json

    ```php
    "rekonekt/rekonekt-php-sdk": "~1.0"
    ```

2. Update packages by running

    ```
    composer update
    ```

3. Use library as shown in [examples]

[examples]: /examples/index.php
